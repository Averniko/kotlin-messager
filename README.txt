Мессенджер.
Приложение позволяет вести общение посредством текстовых сообщений.
Приложение должно позволять пользователю выполнять аутентификацию (регистрацию и авторизацию).
Приложение должно позволять пользователю просматривать, добавлять, редактировать и удалять контакты в приложении.
Приложение должно позволять вести переписку с контактами один на один:
- отправлять сообщения,
- принимать сообщения,
- просматривать историю сообщений.
Приложение позволяет отображать push-уведомления о получении новых сообщений.
